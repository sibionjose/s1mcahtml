JavaScript Predefined Functions

String Object

The String object in JavaScript allows you to work with strings in various ways. Here are some commonly used predefined functions:

  • String.length

    Returns the length of the string.

    let str = "Hello, World!";
console.log(str.length); // 13
  • String.toUpperCase()

    Converts the string to uppercase.

    let str = "hello";
console.log(str.toUpperCase()); // HELLO
  • String.toLowerCase()

    Converts the string to lowercase.

    let str = "HELLO";
console.log(str.toLowerCase()); // hello
  • String.substring(start, end)

    Returns a part of the string between the specified indices.

    let str = "Hello, World!";
console.log(str.substring(0, 5)); // Hello
  • String.split(separator)

    Splits the string into an array of substrings based on the specified separator.

    let str = "apple,banana,cherry";
console.log(str.split(",")); // ['apple', 'banana', 'cherry']

Math Object

The Math object provides various mathematical functions and constants. Here are some commonly used functions:

  • Math.max(...values)

    Returns the largest of the given numbers.

    console.log(Math.max(1, 2, 3)); // 3
  • Math.min(...values)

    Returns the smallest of the given numbers.

    console.log(Math.min(1, 2, 3)); // 1
  • Math.random()

    Returns a random floating-point number between 0 (inclusive) and 1 (exclusive).

    console.log(Math.random()); // e.g., 0.123456789
  • Math.floor(x)

    Rounds down x to the nearest integer.

    console.log(Math.floor(5.7)); // 5
  • Math.ceil(x)

    Rounds up x to the nearest integer.

    console.log(Math.ceil(5.1)); // 6
  • Math.sqrt(x)

    Returns the square root of x.

    console.log(Math.sqrt(16)); // 4
